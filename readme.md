# Image Gallery Block

Image Gallery Block helps you create responsive photo galleries, masonry layouts, and 3D sliders directly in the WordPress Gutenberg editor.

[Image Gallery Block](https://bplugins.com/products/3d-image-gallery/) is a lightweight and powerful WordPress plugin that allows you to create responsive photo galleries, grid layouts, and lightbox popups with ease.

[![Image Gallery Block Banner](https://ps.w.org/3d-image-gallery/assets/banner-772x250.png)](https://bplugins.com/products/3d-image-gallery/)

## 🚀 Key Features – Free Version
- **Responsive Image Gallery Block**: Automatically adapts to desktop, tablet, and mobile screens.
- **Easy Image Gallery Creation**: Quickly create and display photo galleries or photo albums directly inside the Gutenberg editor.
- **Responsive Gallery Layouts**: Choose from Classic Grid and Polaroid styles that automatically adapt to all devices.
- **Built-in Lightbox Popup**: Allow visitors to open images in a clean popup lightbox for a better viewing experience.
- **Flexible Grid Controls**: Adjust gallery columns (1–6) and spacing to create responsive image gallery layouts.
- **Custom Styling Controls**: Design gallery cards using gradients, shadows, and hover effects.
- **Optimized Performance**: Scripts only load when the block is used, helping maintain site performance.

## 🛠️ Installation
1. Log in to your WordPress dashboard.
2. Go to **Plugins > Add New**.
3. Search for "Image Gallery Block".
4. Click **Install Now** and then **Activate**.

## 📖 How to Use
1. Open the Block Editor (Gutenberg) on any Page or Post.
2. Search for the **Image Gallery** block.
3. Use the block settings in the sidebar to add images, set columns, and customize the design.
4. Publish your page and enjoy your professional image gallery!
